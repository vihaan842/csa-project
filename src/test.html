Demo

This is a demonstration file made to showcase the abilities of this browser. It should be located at src/test.html

If you look at the source of this file, you will notice that it is a completely normal HTML web file (static content) with some embedded JavaScript (to make it dynamic). You can edit the file as you please and upon rerunning the browser, the content/behavior of this page should change.

This browser is very, very far from supporting all web features. Regarding the JavaScript interpreter, which was the main focus of this project, it is theoretically capable of doing a lot, but I have not implemented everything. If you stick to the features I used, it should work (note that you cannot use postfix operators like ++, nor can you use less than due to a limitation in my HTML parser). Basic if statements, for loops, while loops, functions, classes, and methods should work.

The interactive element presented here is a button. Upon pressing it, the counter below it should increment. Additionally, it should display and undisplay another element, which will also sometimes display a different message (including a calculation of the counter factorial) in different styling (to show that the style of the element can be changed as well). Additionally, some output is printed in the terminal. There is also a second button which hides the first one. You will not be able to activate the first button after it is hidden.
